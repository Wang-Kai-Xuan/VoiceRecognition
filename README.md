# 项目介绍
这是一个基于百度云语音+PyQt4的一个语音识别软件,就是把语音转换为字符串。
可以识别中文(普通话)，英语

# 开发p平台及环境

```
window10
+
Eclipse IDE for C/C++ Developers
+
pyqt4
+
pydev
+
Anaconda
```
