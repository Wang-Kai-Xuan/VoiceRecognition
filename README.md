# 项目介绍
本项目是一款多媒体的软件。
项目需求：
1. 能够录制声音

2. 能够播放音乐
	1. 播放|暂停功能
	2. 扫描指定目录音乐文件
	3. 显示歌词
	4. 循环播放
	5. 变速播放

3. 能够语音识别
	1. 把语音转换为字符串
	2. 可以识别中文(普通话)
	3. 可以识别英语
